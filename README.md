## Overview

This project performs a complete **Exploratory Data Analysis (EDA)** on the World Happiness Report dataset (2015вЂ“2019) sourced from Kaggle. The goal is to uncover patterns, trends, and insights about what makes nations happy вЂ” and which countries are falling behind.

## Key Findings

- **Finland ranked #1 in 2019** with a happiness score of **7.769**
- **GDP has the strongest correlation** with happiness: **r = 0.789**
- **Generosity has the weakest** correlation: **r = 0.138** вЂ” the most surprising finding
- **Venezuela suffered the largest rank drop:** #23 в†’ #108 (в€’85 places) between 2015 and 2019
- **Global average score changed by only +0.031** over 5 years вЂ” the world is not getting happier
- **57.8%** of all country-year observations fall in the Low (4вЂ“5) or Medium (5вЂ“6) tiers
